Konfigurasi IP

  1. Login menggunakan user root
  2. lalu ketikan nano /etc/network/interfaces
  3. ketikan :

    4. auto enp0s3

    iface enp0s3 inet static

    address 192.168.136.1 (ip bebas)

    netmask 255.255.255.0

  4. lalu save dengan cara ctrl + s
  5. untuk keluar dari konfigurasi ketikan ctrl + x
  6. langkah terakhir merestart ip dengan command /etc/init.d/networking restart

Menambahkan DVD di Debian10

  5. untuk mengecek dvd ketikan nano /etc/apt/sources.list

Instalasi dan Konfigurasi DHCP server

  1. login menggunakan user root
  2. lalu setting ip server terlebih dahulu klik di sini
  3. ketikan apt-get install isc-dhcp-server untuk menginstall dhcp server
  4. masukan dvd 2, klikc disini
  5. kemudian konfigurasi file dhcpd.conf dengan command nano /etc/dhcp/dhcpd.conf seperti gambar di bawha ini
  6. jika sudah save file dengan ctrl + s
  7. untuk keluar dari konfigurasi ketikan ctrl + x
  8. lalu konfigurasi file default/isc-dhcp-server dengan command nano /etc/default/isc-dhcp-server
  9. ketikan enp0s3 dalam INTERFACESv4 INTERFACES="enp0s3"
  10. jika sudah save file dengan ctrl + s
  11. untuk keluar dari konfigurasi ketikan ctrl + x
  12. langkah terakhir merestart dhcp server dengam command /etc/init.d/isc-dhcp-server restart

Instalasi dan Konfigurasi DNS (bind9)

  1. login menggunakan user root
  2. lalu setting ip server terlebih dahulu klik di sini
  3. selanjutnya siapkan DVD 1 dan DVD 2
  4. Lalu install bind9 dengan command apt-get install bind9 -y
  5. masukan dvd yang di butuhkan, jika terdapat pilihan pilih "Force Unmount"
  6. dan jika Instalasi error ketikan, apt --fix-broken install -y
  7. masukan lagi dvd yang di buthkan, jika terdapat pilihan pilih "Force Unmount"
  8. langkah selanjutnya pindahkan direktori ke bind, dengan command cd /etc/bind
  9. lalu kita akan membuat copy file local dengan command cp db.local db.syarief
  10. selanjutnya, konfigurasi file db.syarief dengan command nano db.syarief
  11. jika sudah save file dengan ctrl + s
  12. untuk keluar dari konfigurasi ketikan ctrl + x
  13. lalu kita akan membuat copy file local dengan command cp db.127 db.192
  14. selanjutnya, konfigurasi file db.192 dengan command nano db.192
  15. jika sudah save file dengan ctrl + s
  16. untuk keluar dari konfigurasi ketikan ctrl + x
  17. berikutnya konfigurasi file named.conf.local dengan command nano named.conf.local

    ketikan :

    zone "syarief.com" {
  type master;
  file "/etc/bind/db.syarief";
};

zone "136.168.192.in-addr.arpa" {
    type master;
    file "/etc/bind/db.192";
};
  18. jika sudah save file dengan ctrl + s
  19. untuk keluar dari konfigurasi ketikan ctrl + x
  20. berikutnya konfigurasi file named.conf.options dengan command nano named.conf.options
  21. selanjutnya konfigurasi resolv.conf dengan command nano /etc/resolv.conf

    lalu ketikan

    search syarief.com
nameserver 192.168.136.1
  22. jika sudah save file dengan ctrl + s
  23. untuk keluar dari konfigurasi ketikan ctrl + x
  24. langkah terakhir yaitu merestart dns dengan commmand /etc/init.d/bind9 restart
  25. sebelum mengecek dns rubah ip pada control panel seuai ip server
  26. untuk mengecek dns server install dnsutils dengan command apt-get install dnsutils -y
  27. masukan dvd yang di butuhkan
  28. ketikan nslookup syarief.com atau nslookup 192.168.136.1 bisa juga serach di browser.

Instalasi dan Konfigurasi file server (samba)

  1. login menggunakan user root
  2. lalu setting ip server terlebih dahulu klik di sini
  3. kemudian ketik apt-get install samba -y untuk mengintall file server
  4. masukan DVD yang di butuhkan
  5. untuk memastikan samba telah terinstall ketik ulang apt-get install samba -y
  6. kemudian konfigurasi file smb dengan command nano /etc/samba/smb.conf
  7. scroll hingga paling akhir dan ketikan :

    8. [Sharing]

    path = /home/sahring

    writeable = yes

    read only = yes

    browsable = yes

    public = yes

  8. jika sudah save file dengan ctrl + s
  9. untuk keluar dari konfigurasi ketikan ctrl + x
  10. pindahkan direktori ke home dengan command cd /home
  11. buat file sharing dengan command mkdir sharing
  12. atur access folder dengan command chmod 777 sharing
  13. langkah terakhir restart file server dengan command /etc/init.d/samba-ad-dc restart
  14. untuk mengecek file server berhasil, buka file explorer pergi ke network ketikan ip server
    15. \\192.168.1.1 (contoh)

Konfigurasi Mail Server

  1. Langkah pertama yaitu konfigurasi IP kalian, click here
  2. Pastikan sudah menginstall DNS (Domain Name System) dan tambahkan subdomain Installasi pada file /etc/bind/db.syarief (sesuai yang kalian setting).
    tambahkan seperti ini : 
    3. mail  IN  A   192.168.27.1 (ip server kalian)

    jika sudah simpan dengan cara ctrl + x, y

  3. langkah selanjutnya install mail server dengan command apt install postfix dovecot-imapd dovecot-pop3d -y
  4. akan muncul jendela baru seperti gambar dibawah ini. Pilih Internet Site
  5. Akan muncul jendela baru lagi, Isikan domain yang akan dipakai oleh mail server kita, example : mail.syarief.com
  6. Kemudian buat directory untuk menyimpan mail server kita, dengan perintah: maildirmake.dovecot /etc/skel/Maildir

  7. Lakukan konfigurasi aplikasi postfix dengan mengetikkan perintah nano /etc/postfix/main.cf Pada baris dibawah tambahkan code home_mailbox = Maildir/ seperti gambar di atas
  8. Lakukan perubahan pada file 10-mail.conf dengan mengetikkan perintah nano /etc/dovecot/conf.d/10-mail.conf
    konfigurasi seperti gambar di atas
  9. Lakukan perubahan pada file 10-auth.conf dengan mengetikkan perintah nano /etc/dovecot/conf.d/10-auth.conf
    konfigurasi seperti gambar di atas
  10. Lakukan perubahan pada file dovecot.conf dengan mengetikkan perintah nano /etc/dovecot/dovecot.conf
  11. Kita lakukan konfigurasi kembali pada paket aplikasi postfix dengan mengetikkan perintah dpkg-reconfigure postfix
  12. Kemudian akan muncul jendela baru seperti gambar dibawah ini. Pilih Internet Site
  13. Akan muncul jendela baru lagi, Isikan domain yang akan dipakai oleh mail server kita
  14. Pada bagian Root and Postmaster mail recipient: kita kosongkan saja
  15. Kemudian pastikan subdomain sudah muncul pada jendela berikutnya
  16. Pada bagian “Force synchronous updates on mail queue?” kita pilih “NO”
  17. Pada baris paling akhir, kita isikan Network ID yang digunakan oleh mail server kita.
  18. Pada jendela “Mailbox size limit (bytes)” biarkan 0 lalu enter
  19. Bila muncul perintah, bisa langsung tekan “OK” untuk melanjutkan !
  20. Kita pilih IP Address versi 4 atau ipv4
  21. Jangan lupa untuk merestart semua yang telah di konfigurasi (DNS, POSTFIX, DOVECOT) dengan command :
    - /etc/init.d/bind9 restart
    - /etc/init.d/postfix restart
    - /etc/init.d/dovecot restart

Roundcube

  1. Sebelum melakukan proses installasi webmail roundcube pastikan kalian sudah menginstall mail server terlebih dahulu di server debian click here
  2. Jika sudah membuat mail server pada debian, kemudian install web server apache. Perintah untuk melakukan installasi web server apache adalah apt-get install apache2
  3. Kemudian pastikan sudah menginstall database MariaDB click here
  4. jika MariaDB sudah terinstall masuk ke database dengan command mysql -u root -p
  5. ketikkan perintah seperti di bawah ini :
    1. > create database roundcube;
    2. > create user 'roundcube'@'localhost' identified by '123';
    3. > grant all privileges on roundcube.* to 'roundcube'@'localhost';
    4. > flush privileges;
    jika sudah exit
  6. kemudian install webmail roundcube dengan perintah apt install roundcube -y
  7. Pada saat proses installasi akan ada sebuah jendela pertanyaan configure database for roundcube with dbconfig-common? pilih saja Yes
  8. Kemudian akan ada pertanyaa mysql application password for roundcube isikan password bebas
  9. kemudian akan ada jendela baru lagi Password confirmation masukkan password yang tadi isi
  10. Lakukan konfigurasi roundcube pada file config.inc.php dengan mengetikkan perintah nano /etc/roundcube/config.inc.php
    Lakukan perubahan pada
    $config[‘default_host’] = ‘mail.syarief.com’; --> isi dengan domain servernya
    $config[‘smtp_server’] = ‘mail.syarief.com’; --> isi dengan domain servernya
    $config[‘smtp_user'] = ‘ ’; --> dikosongkan
    $config[‘smtp_pass'] = ‘ ’; --> dikosongkan
  11. Masuk ke directory sites-available dengan perintah cd /etc/apache2/sites-available
    kemudian copy file 000-default.conf menjadi mail.conf dengan perintah cp 000-default.conf mail.conf
  12. Kemudian lakukan konfigurasi pada file mail.conf, dengan mengetikkan perintah nano mail.conf
    lakukan perubahan pada
    …
    ServerName mail.syarief.com --> hapus tanda pagar yang ada di depannya dan subdomain
    DocumentRoot /var/lib/roundcube --> arahkan ke dokumen roundcube
    …
  13. Selanjutnya aktifkan file main.conf dengan perintah a2ensite mail.conf dan restart apache dengan perintah systemctl reload apache2
  14. Kemudian masuk ke browser di pc client dan ketikkan subdomain yang sudah dibuat yaitu mail.syarief.com
  15. sebelum masuk ke browser silahkan buat 2 user baru untuk manjadikan sisi pengirim dan sisi penerima dengan perintah :
    adduser syarief (bebas)
    adduser rangga(bebas)
